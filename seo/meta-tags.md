## Product Page Template

### Title Format
```
[Nombre del Producto] al Por Mayor | [Marca] - Dulcitienda
```
**Example:**
```
Chocolates Jet al Por Mayor | Jet - Dulcitienda
```

### Meta Description Format
```
Compra [Nombre del Producto] al por mayor en Neiva. [Marca] para surtir tu tienda. Precios especiales distribuidores. Envío en Huila.
```
**Example:**
```
Compra Chocolates Jet al por mayor en Neiva. Jet para surtir tu tienda. Precios especiales distribuidores. Envío en Huila.
```

### Product OG Tags
```html
<meta property="og:title" content="[Producto] al Por Mayor | [Marca] - Dulcitienda">
<meta property="og:description" content="[Descripción corta del producto]. Precio mayorista en Neiva.">

<meta property="product:condition" content="new">
```

## Canonical URLs & Hreflang

### Canonical Tags
```html
<!-- Homepage -->
<link rel="canonical" href="https://dulcitienda.com">

<!-- Category pages -->
<link rel="canonical" href="https://dulcitienda.com/categoria/dulces">

<!-- Product pages -->
<link rel="canonical" href="https://dulcitienda.com/producto/[slug]">

<!-- Pagination -->
<link rel="canonical" href="https://dulcitienda.com/categoria/dulces">
<link rel="prev" href="https://dulcitienda.com/categoria/dulces">
<link rel="next" href="https://dulcitienda.com/categoria/dulces?page=2">
```

