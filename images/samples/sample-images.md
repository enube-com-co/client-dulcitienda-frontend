## Usage Notes

### Image Quality Parameters
All URLs include quality parameters:
- `w=800` - Width 800px (adjust as needed: 300, 600, 1200)
- `q=80` - Quality 80% (Unsplash)
- `auto=compress` - Auto compression (Pexels)

### How to Use
```typescript
// Example usage in Next.js
<Image
  src="https://images.unsplash.com/photo-1622483767028-3f66f32aef97?w=800&q=80"
  alt="Coca-Cola Original 350ml"
  width={600}
  height={600}
  onError={(e) => {
    e.currentTarget.src = "fallback-url-here";
  }}
/>
```

### License Information
- **Unsplash**: CC0 - No attribution required, free for commercial use
- **Pexels**: Pexels License - Free for commercial use, attribution appreciated

### Resizing Images
To get different sizes, modify the URL parameters:

**Unsplash:**
- Thumbnail: `?w=300&q=80`
- Medium: `?w=600&q=80`
- Large: `?w=1200&q=80`

**Pexels:**
- Thumbnail: `?auto=compress&w=300`
- Medium: `?auto=compress&w=600`
- Large: `?auto=compress&w=1200`

### Fallback Strategy
Always implement error handling to fall back to a generic placeholder:
```typescript
const handleImageError = (e: React.SyntheticEvent<HTMLImageElement>) => {
  e.currentTarget.src = '/placeholders/product.svg';
};
```
